元组(tuple)

  • 元组的元素无法修改，元组使用小括号 () ，元组是不可变数据类型，不能修 改或删除元素，也不能添加元素。
  • 用()创建元组，元素之间用英文逗号隔开
  • 如果一个元组对象中只有一个元素，那么这个元素后面一定要接一个 , （逗号）不然这个对像就不是元组类型
